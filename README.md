# feedBack-react
simple feedback app using react and json server 

## Run Locally

Clone the project

```bash
  git clone https://github.com/mostafa-kheibary/feedBack-react.git
```

Go to the project directory

```bash
  cd feedBack-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
Start the json server

```bash
  npm start-server
```
